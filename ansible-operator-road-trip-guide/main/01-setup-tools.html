Setup

Resources

  • Access to an OCI compliant image registry where you can push images, I’ll be using quay.io, other registries should work as well.

  • Kubernetes/OpenShift Cluster (where you have administrative permissions) or use Minikube as we’re going to do most of the time in this guide.
